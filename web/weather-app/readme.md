# Погода в місті

Адаптивний веб-застосунок для перегляду поточної погоди та прогнозу на 5 днів у будь-якому місті світу. Розроблено в рамках індивідуального проєкту №1 з дисципліни «WEB-орієнтовані технології. Frontend розробки» (ФІОТ, КПІ, кафедра ІСТ).

### Основні можливості
- Пошук погоди за назвою міста
- Відображення поточної погоди (температура, відчувається, вологість, вітер, опади)
- Прогноз на 5 днів (температура min/max, опис, опади)
- Список улюблених міст (зберігання в localStorage)
- Ручне додавання/видалення улюблених міст
- Повністю адаптивний інтерфейс (десктоп, планшет, смартфон)
- Бургер-меню на мобільних пристроях
- Підтримка прямого доступу за посиланням (наприклад: `.../#kyiv`)

### Технології
- HTML5
- CSS3 (flexbox, grid, media queries, анімації)
- JavaScript (ES6+, async/await, fetch API, DOM-маніпуляції)
- Зовнішній API: Open-Meteo (геокодинг + прогноз погоди)
- Без фреймворків і бібліотек
